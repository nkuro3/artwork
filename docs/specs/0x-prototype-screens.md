## 5. 画面仕様

各画面: 目的 / 主な要素 / 状態 / 遷移 / 主要文言。共通事項（ヘッダー・状態パターン）は `02` §5 を参照し再掲しない。

### 5.1 ホーム `/`
| 項目 | 内容 |
|---|---|
| 目的 | サービスの入口。未ログインは簡易案内、ログイン済みは管理へ誘導 |
| 挙動 | ログイン済みは `/artworks` へリダイレクト。未ログインはランディング表示 |
| 要素（未ログイン） | サービス名（h1）/ 一言説明 / `ログイン`・`登録` ボタン |
| 文言 | 見出し「アートワーク」 / 説明「作品を登録して公開ポートフォリオを作る」 |

### 5.2 ログイン `/login`
| 項目 | 内容 |
|---|---|
| 目的 | 既存ユーザーのログイン |
| 要素 | h1「ログイン」/ メール入力 / パスワード入力 / `ログイン`ボタン / `サインアップへ`リンク |
| 状態 | 入力エラー（形式/必須）はフィールド近傍 / 認証失敗は `role="alert"`「メールまたはパスワードが違います」 |
| 遷移 | 成功 → `/artworks` / 「サインアップへ」→ `/signup` |

### 5.3 サインアップ `/signup`
| 項目 | 内容 |
|---|---|
| 目的 | 新規登録（直後に artist_profile 初期化＝FR-03） |
| 要素 | h1「登録」/ 表示名 / メール / パスワード(8文字以上) / `登録`ボタン / `ログインへ`リンク |
| 状態 | バリデーション（メール形式・パスワード長・必須）/ 重複メールは `role="alert"`「このメールは登録済みです」 |
| 遷移 | 成功 → `/artworks` / 「ログインへ」→ `/login` |

### 5.4 ログアウト `/logout`
| 項目 | 内容 |
|---|---|
| 目的 | セッション終了 |
| 要素 | 確認テキスト + `ログアウト`ボタン（または即時処理） |
| 遷移 | 実行後 `/login` |

### 5.5 作品一覧 / 管理 `/artworks`（要ログイン）
| 項目 | 内容 |
|---|---|
| 目的 | 自分の全作品の一覧と管理導線（draft/published/archived 含む） |
| 要素 | h1「作品管理」/ `新規作成`リンク / 作品リスト（各項目: サムネ・タイトル・状態(下書き/公開/アーカイブ)・`編集`・`削除`）。**非公開（draft/archived）はカード右上に状態バッジ（`draft`/`archived`）を表示** |
| レイアウト | `02` §4 グリッド列数（`<md`=縦リスト / `≥md`=2〜3列） |
| 状態 | 空: 「作品がまだありません。新規作成」 / 削除中: 当該項目を操作不可に |
| 遷移 | 新規作成→`/artworks/new`（下書き作成し編集へ） / 編集→`/artworks/edit/[id]` / 削除→確認後その場で除去（R2画像も削除＝FR-07） |

### 5.6 作品作成 `/artworks/new`（要ログイン）
| 項目 | 内容 |
|---|---|
| 目的 | 新規下書きの作成と編集画面への遷移 |
| 挙動 | アクセス時に作品（status=draft・タイトル空）を1件作成し、`/artworks/edit/[id]` へリダイレクト。フォームは持たない（編集画面に集約）。新規作成は常に新しい下書きを作る |

### 5.7 作品編集 `/artworks/edit/[id]`（要ログイン）
| 項目 | 内容 |
|---|---|
| 目的 | 作品の編集（所有者のみ）。状態(status)の変更もここで行う |
| 要素 | h1「作品を編集」/ タイトル / 説明 / 状態(下書き/公開/アーカイブ select) / 画像アップロード(複数可・並び替え) / `一覧へ戻る`。**掲載チェックボックス・保存ボタンは無し（自動保存）** |
| 自動保存 | タイトル・説明・状態(status)は変更（blur/change）時に PATCH で随時保存。画像は選択時に保存（NFR-02） |
| 状態 | 下書きはタイトル空可。status を `published` にする時はタイトル必須（空なら不可・フィールドエラー）。他人の作品/存在しない → 404「作品が見つかりません」 |
| 遷移 | `一覧へ戻る` → `/artworks`（変更は自動保存済み） |

### 5.8 設定 `/settings`（要ログイン）
| 項目 | 内容 |
|---|---|
| 目的 | プロフィールと Profile Slug（公開URLハンドル）の編集 |
| 要素 | h1「設定」/ 表示名 / **Profile Slug**（入力は `@` プレフィックス表示・保持値に `@` は含めない・重複/形式チェック）/ 自己紹介(bio) / `保存`ボタン / `ポートフォリオ編集`(/portfolio/edit) と 自分のページ（`@{slug}` 表記で `/p/{slug}` へのリンク） |
| 状態 | slug 重複は `role="alert"`「この slug は使用されています」/ 形式不正はフィールドエラー |
| 遷移 | 保存 → 同画面で結果表示（slug 変更時は新URL を案内）/ `ポートフォリオ編集` → `/portfolio/edit` |

公開制御は作品の `status`（published＝公開）で行う。ポートフォリオ掲載は `/portfolio/edit`（§5.12）で選ぶ。

### 5.9 検索結果 `/search`（公開）
| 項目 | 内容 |
|---|---|
| 目的 | 作品・作者の横断検索（FR-17、pg_trgm 部分一致） |
| 検索対象 | 公開作品（`status='published'`、title/description の部分一致）と 公開プロフィール（display_name）。draft/archived は対象外 |
| 要素 | 検索ボックス（クエリ `q`）/ `作品` 結果（サムネ+タイトル、`/p/{slug}/{id}` へ）/ `作者` 結果（表示名+slug、`/p/{slug}` へ） |
| レイアウト | `02` §4 グリッド列数（作品=`≥md` で2列、作者=リスト） |
| 状態 | 空クエリ: 結果非表示・プロンプト「キーワードを入力」/ 0件: 「該当がありません」 |
| 遷移 | ヘッダーの `検索` リンク（全状態で表示）or 直接アクセス。結果クリック → 各公開ページ |

### 5.10 公開ポートフォリオ `/p/[slug]`（公開・未認証）
| 項目 | 内容 |
|---|---|
| 目的 | 作者のポートフォリオ（選んだ公開作品、SSR + キャッシュ、NFR-06） |
| 要素 | h1「作者表示名」/ `@{slug}`（Profile Slug）/ 自己紹介(bio, 任意) / ポートフォリオ作品グリッド（サムネ+タイトル、`/p/{slug}/{id}` へ） |
| 表示条件 | `status='published'` かつ **ポートフォリオに選択された作品**・選択順（position）。§5.12 で編集（FR-12,13） |
| レイアウト | `02` §4 グリッド列数（`<md`=1〜2列 / `≥md`=3列） |
| 状態 | slug 該当なし → 404「ポートフォリオが見つかりません」/ 公開作品0件 → 「公開作品はまだありません」 |
| SEO | title=表示名、OGP 画像=先頭作品の先頭画像（FR-16） |

### 5.11 公開作品詳細 `/p/[slug]/[artworkId]`（公開・未認証）
| 項目 | 内容 |
|---|---|
| 目的 | 公開作品1点の詳細閲覧 |
| 表示条件 | `status='published'`（公開）なら閲覧可。ポートフォリオ未選択でも published なら直リンク・検索から到達できる |
| 要素 | h1「作品タイトル」/ 説明 / 画像（詳細用・大サイズ、複数は縦並び）/ `{作者}のポートフォリオへ戻る`リンク |
| 状態 | 非公開（draft/archived）/存在しない → 404「作品が見つかりません」 |
| SEO | title=「作品名 - 作者名」、OGP=先頭画像 |

### 5.12 ポートフォリオ編集 `/portfolio/edit`（要ログイン）
| 項目 | 内容 |
|---|---|
| 目的 | 公開ポートフォリオ（`/p/[slug]`）に載せる作品の選択と並び替え（1人1ポートフォリオ） |
| 要素 | h1「ポートフォリオ編集」/ 自分の**公開（published）作品**のリスト（各: サムネ・タイトル・`掲載`チェック・並び替え `↑`/`↓`）/ `保存`ボタン / 公開ページ(`/p/{slug}`) と 設定 へのリンク |
| 振る舞い | 掲載チェックで選択、`↑`/`↓` で選択内の表示順を変更。`保存`で選択集合＋順序を確定（公開ページに反映） |
| 状態 | 公開作品が0件: 「公開作品がありません。作品を公開すると掲載できます」（`/artworks` への導線） |
| 遷移 | 保存 → 同画面で結果反映 / `公開ページを見る` → `/p/{slug}` |
